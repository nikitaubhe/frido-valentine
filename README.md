🛍️ Shopify Assignments – Valentine’s Landing Page

Assignment 2: Valentine’s Day Landing Page

🔹 Overview
An interactive landing page with dual modes (Couple & Single) and dynamic product filtering.

✨ Key Features

🔁 Mode Toggle Animation
- Toggle between:
  - **Couple Mode** (Red/Warm Theme)
  - **Single Mode** (Dark Theme)
- Smooth UI transition (background, text, video)

🎥 Dynamic Hero Section
- Couple Mode → Romantic banner video  
- Single Mode → Independent vibe banner video  
- Headline and CTA update dynamically

🛍️ Product Cards (Shopify API)
- Products fetched using **Shopify Storefront GraphQL API**
- No hardcoded data
- Each card includes:
  - Image
  - Product name
  - Original price
  - Sale price
  - Discount badge
  - CTA button

🗂️ Category Filtering
Couple Mode:
- Sleep  
- Travel  
- WFH  
- GYM  
Single Mode:
- Mom  
- Dad  
- BFF  
- Kids  

- Filtering is implemented using **product tags**
 🧠 Tech Stack
- HTML  
- CSS (Custom + Animations)  
- JavaScript   
- Shopify Storefront API (GraphQL)

🔗 API Used
- Shopify Storefront API
